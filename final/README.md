# Final Project Code

## Requirements

Make sure you have the following Python packages installed:
- `numpy`
- `matplotlib`

You can install them using py pip install, which should be upgraded as well.

## How to run the code:
Run the following command for the 
Triangle Lattice Structure:
```
python3 tri_lat_sim.py 
```

Star-Grid Lattice Structure:
```
python3 star_lat_sim.py 
```

Bowtie Lattice Structure:
```
python3 bow_lat_sim.py 
```
